Detailed Description

Static user configuration data which must survive reboot and firmware update. Do not change the field order, due to firewall constraint the tls_device_key size must be placed at a 64 bit boundary. Its size sould also be multiple of 64 bits.

Depending on the available board peripherals, the c2c_config and wifi_config fields may not be used.
