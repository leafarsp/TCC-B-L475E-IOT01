62/* Socket options definitions:
63 * Important: All contents are passed by reference.
64 * The callee does not make a copy.
65 * The caller is responsible for maintaining the passed memory area valid until the socket is destroyed.
66 *
67 * Name Content Effect
68 * tls_ca_certs List of the root CA cerficates. PEM format. TLS lib configuration.
69 * tls_ca_crl Certificate revocation list. PEM format. TLS lib configuration.
70 * tls_dev_cert Client certificate. PEM format. TLS lib configuration.
71 * tls_dev_key Client private key. PEM format. TLS lib configuration.
72 * tls_dev_pwd Client private key password. Binary format. TLS lib configuration.
73 * tls_server_verification NULL TLS lib configuration.
74 * tls_server_noverification NULL TLS lib configuration.
75 * Default option: tls_server_verification
76 *
77 * tls_server_name Check pattern for the server certificate verification. String. TLS lib configuration.
78 * sock_blocking NULL. The recv calls are blocking until
79 * - at least one byte may be returned,
80 * - or the sock_read_timeout is reached.
81 * sock_noblocking NULL. The recv calls are not blocking.
82 * The available payload is immediately returned, even if empty.
83 * Default option: sock_blocking
84 *
85 * sock_read_timeout Timeout in ms. Ascii format.
86 * sock_write_timeout Timeout in ms. Ascii format. Applied to TCP sockets only.
87 */
