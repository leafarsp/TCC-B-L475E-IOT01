Read from a socket.

Note
If the "sock_blocking" option was set, the function will not return until the requested length is received, or the "sock_read_timeout" is reached. If the "sock_noblocking" option was set, the function will return immediately up to the requested length.
Parameters
Insockhnd Socket.
Outbuf Destination buffer. Allocated by the caller.
Inlen Length to be read.
Return values
Status>=0 Success. Number of bytes read. NET_TIMEOUT In "sock_blocking" mode, the read timeout was reached. NET_EOF The connection was closed. NET_ERR Internal error. NET_PARAM Invalid parameter passed.

◆ net_sock_setopt()

int net_sock_setopt ( net_sockhnd_t  sockhnd,
const char *  optname,
const uint8_t *  optbuf,
size_t  optlen 
)

Set a socket option.

Note
May be called before the socket is opened.
The function does not copy the option contents to the context storage: The caller must keep the data available at their passed location until the socket is closed.
Parameters
Insockhnd Socket.
Inoptname Option name.
Inoptbuf Option payload.
Inoptlen Option payload length. If the data is an ASCII string, the length must include the \0 termination char.
Return values
StatusNET_OK Success. NET_PARAM The option is not supported, or the contents are inconsistent with the option. NET_ERR Internal error. NET_AUTH The remote host cound not be authentified. (Applicable to NET_PROTO_TLS sockets).

References NET_PARAM, and net_sock_ctxt_s::proto.

Referenced by http_sock_setopt().
