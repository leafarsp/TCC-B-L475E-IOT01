Read from an HTTP progressive download session.

Parameters
Outreadbuffer Output buffer.
Outstatus HTTP range download status.
Inoffset Offset (in bytes) from the start of the remote resource to read from. (Supported only with GET requests).
Insize Size of the chunk to read.
Inextra_headers String containing additional HTTP headers to send. Each line must end with \r
. "" for no header at all.
Inpost_buf Payload of the POST request. If NULL, the call is translated into a GET request.
Inpost_buf_size Size of the POST payload.
Inhnd Session handle.
Return values
>=0Success: Number of bytes copied to readbuffer. <0 Error: HTTP_ERR Bad input parameter. Or allocation error. HTTP_ERR_HTTP Connection error, or unexpected response from the HTTP server. HTTP_ERR_CLOSED The connection was closed by the server during the previous http_read(). It is allowed by the protocol.

Create an HTTP request string from the parameter list.

Note
The request buffer is allocated by the callee and must be released by calling http_req_destroy() after usage.
Parameters
Outreq_buf Output buffer, allocated by the callee.
Inquery HTTP query string.
Inhostname Remote hostname.
Inoffset Offset to the first byte to be read from the requested document in case of a partial content request. Must be set to 0 otherwise.
Insize Maximum length of the chunk of the requested document to be read.
Inextra_headers HTTP headers to be appended to the request header. Single buffer. Each line must be ended by \r
.
Inpost_buf_size Size of the document body to be sent, in case of a POST request. Must be 0 for a GET request.
Return values
Errorcode >0 Success, length of the request string written to req_buf. HTTP_ERR (<0) Failure
